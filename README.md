# Cartes thématiques de la croissance démographique à Niakhar

## Description
Cartographie de la croissance démographique dans la zone de Niakhar entre 2004 et 2012.

## Méthodologie
- Logiciel : QGIS
- Données : population par zone
- Thématique : taux de croissance par zone
